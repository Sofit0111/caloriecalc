# 🍏 Калории Бот

Telegram бот для отслеживания калорийности пищи с поддержкой различных режимов (похудение, поддержание, набор).

## ✨ Функции

- 📝 **Записать прием пищи** - добавление продуктов с указанием веса
- 📊 **Статистика** - графики калорий за период (1, 3, 7, 30 дней, или свой)
- ⚙️ **Настройки** - установка цели калорий и выбор режима
- ⭐ **Избранное** - сохранение часто используемых продуктов
- 👑 **Админ-панель** - управление продуктами, рассылка, статистика
- 🔍 **Поиск** - быстрый поиск продуктов по названию (FTS5)
